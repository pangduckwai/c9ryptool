## Example
```bash
> c9ryptool decode -i cek.jwe -o cek.sec
> c9ryptool decode -i ctnt.jwe -o ctnt.sec
> c9ryptool decode -i iv.jwe -o iv.bin
> c9ryptool decode -i tag.jwe -o tag.bin
> c9ryptool decrypt -a RSA-2048-OAEP-SHA256 -k ../prv.key -i cek.sec -o cek.key
> c9ryptool decrypt -a AES-256-GCM -k cek.key -i ctnt.sec --iv=iv.bin --tag=tag.bin --aad=aad.jwe -o jws.out
```
